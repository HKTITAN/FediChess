# FediChess

Decentralized P2P chess in the browser. No central server; peers discover each other via WebTorrent trackers and communicate over WebRTC.

- **Project rules**: `.cursor/rules/` (project context, design, stack, conventions, protocol-and-docs).
- **Skills**: `.cursor/skills/` (fedichess-design, protocol-sync, game-state; plus frontend and Next/React skills when copied from `.agents/skills/`).
- **Documentation index**: [documentation/README.md](documentation/README.md).

Use the path alias `@/` for imports. FediChess design principles (simplicity, fluidity, delight) are the project design language.
